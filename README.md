### Install from source

Open terminal (mac)/command prompt (windows) and type:

    git clone https://github.com/eslrgs/skynthetic.git
    
Navigate to file:

    cd skynthetic
    
Install package:

    pip install .
    
Open jupyter

    jupyter notebook
    
Open 'skynthetic.ipynb' to run notebook
